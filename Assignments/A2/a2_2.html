Q8) Using the iris.csv dataset posted on Quercus, perform the following statistical tests. The “iris” dataset includes the measurement (cm) of length and width of sepal and petal for three species.

  1. One-Sample t-Test

  • Write the null and alternative hypothesis

  • Choose the variable sepal_length and test if its mean is significantly different from a given population mean (5.8 cm)

  • Report the p-value and interpret whether the mean significantly differs from the assumed value (alpha = 0.05)

  • Check if this variable has nearly normal distribution by creating a histogram.

    Null hypothesis (H₀): μ = 5.8 cm (The mean sepal length equals 5.8 cm)

    Alternative hypothesis (H₁): μ ≠ 5.8 cm (The mean sepal length differs from 5.8 cm)

# Perform one-sample t-test for sepal_length against population mean of 5.8

# Separate data for the three species
virginica = iris[iris['species'] == 'virginica']
versicolor = iris[iris['species'] == 'versicolor']
setosa = iris[iris['species'] == 'setosa']

#  Perform ANOVA test to compare means across three species
f_value, p_value = stats.f_oneway(
    setosa['sepal_length '],
    versicolor['sepal_length '],
    virginica['sepal_length ']
)

# Create table of summary statistics
summary_table = pd.DataFrame({
    'Species': ['Setosa', 'Versicolor', 'Virginica'],
    'Mean': [round(setosa['sepal_length '].mean(), 3), 
             round(versicolor['sepal_length '].mean(), 3), 
             round(virginica['sepal_length '].mean(), 3)],
    'Std Dev': [round(setosa['sepal_length '].std(), 3), 
                round(versicolor['sepal_length '].std(), 3), 
                round(virginica['sepal_length '].std(), 3)]
})
